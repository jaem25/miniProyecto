# miniProyecto
Sistema de Gestión de Inventario para una Tienda de Videojuegos

# Descripción
Una tienda de videojuegos necesita un sistema sencillo que permita administrar su inventario de productos. Actualmente el control se realiza en papel, lo que provoca errores al registrar ventas y consultar existencias.
Se solicita desarrollar una aplicación en C++ que permita gestionar los videojuegos disponibles en la tienda mediante un menú interactivo en consola.
El sistema deberá almacenar información básica de cada videojuego, permitiendo registrar nuevos productos, consultar el inventario y actualizar las existencias cuando se realicen ventas o lleguen nuevas unidades.

# Objetivos
Aplicar los conceptos fundamentales de programación estudiados durante el curso:
Variables y tipos de datos.
-Estructuras (struct).
-Arreglos.
-Funciones.
-Sentencias condicionales (if, else).
-Estructuras de selección (switch).
-Ciclos (for, while, do-while).
-Modularización mediante múltiples archivos fuente.

# Requerimientos del Sistema
Cada videojuego deberá almacenar la siguiente información:
  -Código del videojuego.
  -Nombre del videojuego.
  -Precio.
  -Cantidad disponible en inventario.
El sistema deberá permitir almacenar hasta 100 videojuegos.

# Funcionalidades

Las funcionalidades del programa son:
1. Registrar videojuego
2. Mostrar inventario
3. Buscar videojuego
4. Registrar venta
5. Reabastecer inventario
6. Mostrar estadísticas
7. Salir
